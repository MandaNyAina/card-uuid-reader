APPLICATION ANDROID - LECTEUR NFC POUR CARTES MIFARE
====================================================

DESCRIPTION
-----------
Application Android native en Kotlin permettant de scanner les cartes NFC MIFARE Classic S51K 
et d'enregistrer les UIDs dans des fichiers JSON accessibles via ADB ou explorateur de fichiers.

FONCTIONNALITES PRINCIPALES
---------------------------
1. Deux modes de scan:
   - Mode individuel: Chaque carte scannée crée son propre fichier JSON
   - Mode batch: Session de scan multiple avec un fichier JSON pour tout le lot

2. Détection automatique des cartes NFC
3. Extraction et affichage de l'UID
4. Stockage JSON structuré avec métadonnées
5. Interface utilisateur simple et intuitive
6. Compatible avec tous les outils ADB

FICHIERS GENERES
-----------------
Répertoire: /sdcard/Android/data/com.cardreader.app/files/scans/

Mode individuel: scan_YYYYMMDD_HHMMSS_mmm.json
Mode batch: batch_YYYYMMDD_HHMMSS.json

COMPILATION
-----------
Voir INSTRUCTIONS_COMPILATION.txt pour les détails complets

Méthode rapide:
1. Ouvrir dans Android Studio
2. Connecter appareil Android avec NFC
3. Run (Shift+F10)

UTILISATION
-----------
1. Lancer l'application
2. Choisir le mode (individuel ou batch)
3. Si mode batch: cliquer "Démarrer session"
4. Approcher les cartes du téléphone
5. Si mode batch: cliquer "Terminer session"
6. Les fichiers JSON sont créés automatiquement

RECUPERATION DES DONNEES
------------------------
Avec ADB:
adb pull /sdcard/Android/data/com.cardreader.app/files/scans/ ./export/

Sans ADB:
Utiliser un explorateur de fichiers Android pour naviguer vers le dossier

COMPATIBILITE
-------------
- Android 5.0 (API 21) et supérieur
- Appareil avec puce NFC obligatoire
- Testé avec cartes MIFARE Classic S51K
- Compatible avec autres cartes MIFARE et NFC Type A

PERMISSIONS REQUISES
--------------------
- NFC: Lecture des tags NFC
- WRITE_EXTERNAL_STORAGE: Stockage fichiers (Android <10 uniquement)
